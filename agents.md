# AI Agents Guide

This file provides context and instructions for AI agents working on the **djpc.org.uk** project.

## Project Overview
This is a personal website/portfolio built with **Nuxt 3**, **Tailwind CSS**, and **TypeScript**.

## Tech Stack
- **Framework:** Nuxt 3
- **Styling:** Tailwind CSS
- **Language:** TypeScript
- **Package Manager:** Yarn (Berry)

## Directory Structure
- `app/`: Contains the main application code (Vue components, layouts, pages).
- `public/`: Static assets like images and robots.txt.
- `server/`: Server-side logic and API routes.

## Development Workflow
- Follow Nuxt 3 best practices.
- Ensure type safety with TypeScript.
- Use Tailwind CSS for styling.
